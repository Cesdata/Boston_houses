revisão de Preços de Casas em Boston (Regressão)
Este projeto utiliza técnicas de Machine Learning para prever o valor de imóveis em Boston com base em diversas características socioeconómicas e estruturais. O objetivo principal foi demonstrar a criação de um fluxo de trabalho (pipeline) robusto e automatizado.

📋 Descrição do Projeto
O notebook aborda um problema clássico de regressão, focando-se na limpeza de dados e na eficiência do modelo. Foram aplicadas técnicas avançadas de pré-processamento para garantir que o modelo seja facilmente replicável e livre de erros comuns, como o data leakage.

🛠️ Tecnologias e Bibliotecas
Python 3

Pandas & Numpy: Manipulação e análise de dados.

Scikit-Learn: Criação do modelo de regressão, ColumnTransformer e Pipeline.

Category Encoders: Tratamento eficiente de variáveis categóricas.
